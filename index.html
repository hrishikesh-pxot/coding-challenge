In-Place Image Rotation

You are given a grayscale image represented as an N×N 2D list of pixel values (integers 0–255).

Write a function rotate_image(image) that rotates the image 90 degrees clockwise in-place, meaning you cannot create a new 2D list, you must modify the original.

Example

Input
123
456
789
→
Output (90° clockwise)
741
852
963

Rules

  • Modify the image in-place (no new 2D list)
  • No external libraries (no NumPy)
  • Return the rotated image

Constraints

  • Image is always square (N×N)
  • 1 ≤ N ≤ 100
  • Pixel values are integers 0–255

Tests

Your code will be tested against 5 test cases: 3×3, 4×4, 1×1, 2×2, and 5×5 matrices. Click Run Tests to check your solution.
